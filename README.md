# 🧠 Quiz Application (C++)

A console-based Quiz Application built using C++.

## Features
- Multiple choice questions
- Score tracking
- Difficulty levels (Easy & Hard)
- 10-second timer per question

## Concepts Used
- Structures
- Vectors
- Functions
- Time handling
- Conditional logic

## How to Run

Compile:
g++ main.cpp -o quiz

Run:
./quiz
